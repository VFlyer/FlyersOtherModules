On the Subject of Memoryception

You put a memory module within a memory module? How does logic work that way?

This module consists of 3 small displays, 1 large display, 4 stage indicators, and a total of 9 labels consisting of digits 1 - 3. To disarm the module, complete 4 stages on the larger version by completing 3 stages on the smaller version.

This module is divided into 2 sections, the Combined Memory and the Mini Memory. Completing a stage on the Combined Memory will change the displays AND labels on all of the smaller memory modules, alongside resetting progress on the smaller versions if the module is not solved. When interacting with the smaller memory modules, ALL 3 of the smaller displays may change to respect the larger version when completing a stage in the smaller versions`. Note down the initial displays for the larger version alongside the position of the Mini Memory that was solved for each stage on the larger version until the last stage on the larger one. Positions are labeled 1 - 3 from left to right, and are indicated with directions left, middle, and right respectively, for the labels of the Mini Memory modules, and the Mini Memory itself for the Combined Memory.

Failing to correctly interact with this module will result in a strike. If the larger version is incorrectly interacted, it will reset progress and forget the positions interacted and displays on the larger version. If the smaller version is incorrectly interacted, only the smaller version will reset progress, forget the labels and positions pressed, and revert the smaller displays for that stage on the larger display.

Combined Memory

Use the larger displayed digit to determine what Mini Memory to start solving for each stage. For the special cases, if any of the special cases are met, do the special cases instead.

Stage 1

If the displayed digit is a 1, solve the left Mini Memory.

If the displayed digit is a 2, solve the center Mini Memory.

If the displayed digit is a 3, solve the right Mini Memory.

Stage 2

Special Case: If 2 different numbers are shown instead of a number, perform the rule on this stage that corresponds to the other number being displayed. (I.E. 21 are displayed, perform the rule as if the display was a 3.)

If the displayed digit is a 1, solve the Mini Memory whose position is the position of the solved Mini Memory on stage 1.

If the displayed digit is a 2, solve the Mini Memory that had the display initially displaying 3 on stage 1.

If the displayed digit is a 3, solve the Mini Memory whose initial display is the initial display on the Mini Memory solved on stage 1.

Stage 3

Special Case: If 2 different letters are shown instead of a number, solve the Mini Memory that does NOT correspond to the unabbreviated position. (L for left, M for middle, R for right)

Special Case: If a letter is shown instead of a number, solve the Mini Memory corresponding to the unabbreviated position. (L for left, M for middle, R for right)

If the displayed digit is a 1, solve the Mini Memory whose position corresponds to the position of the Mini Memory solved on stage 2.

If the displayed digit is a 2, solve the Mini Memory whose initial display is a 2.

If the displayed digit is a 3, solve the Mini Memory whose initial display is the position of the Mini Memory solved on stage 1.
